# escalante-frontend

## Instalation

```sh
# install package yo
npm install -g yo grunt-cli bower
# Install AngularJS generator
npm install --global generator-angular@0.11.1
# create a project folder
mkdir escalante; cd escalante
# create a angular project
yo angular
# clone the project
git clone https://github.com/mirandarfsm/escalante.git
# start the grunt server
grunt serve
```
